# Rock Paper Scissors Game (Python)

A simple Rock Paper Scissors game built using basic Python concepts such as
if-elif-else, functions, loops, and the random module.

Created by **Utsav Kumar Jha**

---

## About the Project

This is a beginner-friendly Python project where the user plays
Rock Paper Scissors against the computer.

The user chooses rock, paper, or scissors.
The computer randomly selects its choice.
The winner is decided based on standard game rules.
The game keeps track of the score and allows multiple rounds.

This project is made for learning and practice purposes.

---

## Game Rules

- Rock beats Scissors  
- Scissors beats Paper  
- Paper beats Rock  
- Same choice results in a tie  

---

## Concepts Used

- if, elif, else statements  
- Functions  
- while loop  
- input() and print()  
- random module  
- Basic variables and logic
